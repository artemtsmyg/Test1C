Вывести таблицу значений функции y = -0.23x2 + x в текстовое поле. Значения аргумента (x) задаются минимумом, максимумом и шагом. Например, если минимум задан как 1, максимум равен 3, а шаг 0.5. То надо вывести на экран изменение x от 1 до 3 с шагом 0.5 (1, 1.5, 2, 2.5, 3) и значения функции (y) при каждом значении x. Получить с клавиатуры значения x1 (минимум x), x2 (максимум x) и шаг (шаг). Пока x1 не превысит x2 выполнять следующие действия: - вычислить значение y, - вывести на экран текущие значения x1 и y, - увеличить значение x1 на значение шага. - шаг не должен быть отрицательный Моё решение: Процедура ВывестиТаблицуЗначенийФункции() // Ввод значений минимума, максимума и шага x1 = ВводЧисла("Введите минимум x (x1):"); x2 = ВводЧисла("Введите максимум x (x2):"); Шаг = ВводЧисла("Введите шаг:"); // Проверяем, чтобы шаг не был отрицательным Если Шаг <= 0 Тогда Сообщить("Шаг должен быть положительным!"); Возврат; КонецЕсли; // Вывод заголовка таблицы Сообщить(" x | y"); Сообщить("---------------"); // Цикл, который будет выполняться, пока x1 не превысит x2 Пока x1 <= x2 Цикл // Вычисление значения функции y y = -0.23 * x1 * x1 + x1; // Вывод текущих значений x1 и y Сообщить(Формат(x1, "0.00") + " | " + Формат(y, "0.00")); // Увеличиваем значение x1 на шаг x1 = x1 + Шаг; КонецЦикла; КонецПроцедуры // Вспомогательная процедура для ввода чисел с проверкой Функция ВводЧисла(Сообщение) Возврат Строка(Ввод(Сообщение)); КонецФункции
